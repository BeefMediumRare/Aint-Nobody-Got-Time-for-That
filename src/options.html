Track repositories

Add a public GitHub repository folder to index its tracks. Synced tracks are read-only and appear next to your own tracks on the matching videos. The track list is fetched on add and refresh; each track's file is downloaded the first time you open its video, then cached.

    Add a GitHub repository

    This reads every subfolder, however deep, pulling in all track files in the repository. On a large repo that's a lot to download — best reserved for repos dedicated to tracks.
