Add a GitHub repository

Reads every subfolder and downloads all track files in the repo. That's a lot on a big repo, so it's best kept for repos that are only tracks.

Track repositories

Index tracks from a public GitHub folder. Synced tracks are read-only and sit next to yours on matching videos. The list updates on add and refresh; each track downloads the first time you open its video, then caches.
